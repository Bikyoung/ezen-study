폼은 웹에서 사용자의 정보를 입력받기 위한 태그, 또는 사용자가 입력한 데이터를 서버로 보낼 때 사용

label 태그는 입력받는 필드를 지정할 때 사용

input 태그는 데이터를 입력 받을 수 있음

input 태그는 type 속성을 통해 다양한 타입의 데이터를 입려 받을 수 있음. 입력된 데이터는 유효성 검사를 통해 해당 타입에 맞는 데이터인지 확인

fieldset 태그는 폼 요소 안에 있는 데이터를 그룹화할 때 사용. 웹 접근성 준수를 위해 구역을 나누어 줄 때 사용

legend 태그는 fieldset과 함께 사용되며, fieldset의 제목을 달아주는 태그

입력을 필수로 할 때, 해당 태그에 required 속성명만 주면 됨

비활성화하고 싶을 때, 해당 입력 태그에 disabled 속성명만 주면 됨, 또한 서버에 데이터가 전송되지 않음

읽기 전용으로 하고 싶을 때 해당 태그에 readonly 속성명만 주면 됨

첫 진입 시에 포커스를 주고 싶은 태그에 autofocus 속성명만 주면 됨, 사용자가 다른 입력 태그를 클릭하면 포커싱은 종료됨

placeholder : 입력 태그에 hint를 줄 때 사용하는 태그

input 태그의 value 속성 값은 서버에 전송됨

일반 입력 태그의 최대, 최소 글자 크기를 지정할 때는 maxlength와 minlength 속성을 사용

form 태그 속성

  • required : 필수 입력할 필드를 지정할 때 사용
  • readonly : 읽기 전용 필드를 지정할 때 사용. disabled와 달리 입력되어있는 값이 서버에 전송됨
  • disabled : 비활성화 필드를 지정할 때 사용. 서버에 전송 X
  • placeholder : 입력 필드에 hint를 줄 때 사용
  • autofocus : 페이지 로드 시 자동으로 포커스를 줄 필드에 사용
